# FlowDesk

Sistema corporativo de gerenciamento de chamados, desenvolvido para simular um ambiente real de suporte e atendimento técnico.

## Tecnologias

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Angular
- Docker
- GitHub Actions

## Funcionalidades planejadas

- Autenticação de usuários
- Perfis de acesso
- Abertura e acompanhamento de chamados
- Atribuição de técnicos
- Controle de status e prioridade
- Comentários e histórico
- Dashboard com métricas
- Auditoria de alterações
